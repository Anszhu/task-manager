# Team Task Manager Demo Script

Target length: 2-5 minutes

1. Start on the login screen and explain that the first signup becomes the admin automatically.
2. Sign up or log in as the admin and show the dashboard summary cards updating in real time.
3. Open Projects and create a new project with a short description.
4. Add an existing member to the project and show the member list update immediately.
5. Open Tasks and create a task with a deadline and assignee.
6. Click the task row as an admin and show that the task can be edited or deleted.
7. Log out and sign in as the member account.
8. Show that the member can see assigned projects and update only their own task status.
9. Return to Dashboard and show the completed/pending/overdue counts reflecting the task changes.
10. End with the README open to highlight setup instructions, API coverage, and deployment readiness.
